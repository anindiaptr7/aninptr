Materi Hosting

Hosting merupakan layanan yang menyediakan ruang di server agar website dapat diakses melalui internet. Ini adalah konsep vital dalam ASJ.

  • Shared Hosting

    Satu server dibagi-bagi untuk banyak pengguna. Biaya murah, cocok untuk website pemula/kecil.

  • Virtual Private Server (VPS)

    Pembagian server secara virtual, memberikan sumber daya khusus. Lebih fleksibel dan powerful.

  • Dedicated Server

    Satu server fisik penuh hanya untuk satu pengguna. Performa tinggi, kontrol penuh, cocok untuk perusahaan besar.

  • Cloud Hosting

    Menggunakan jaringan server yang saling terhubung. Sangat skalabel, dapat menyesuaikan beban lalu lintas secara dinamis.
