Materi Hosting

WEB Hosting
Hosting merupakan layanan yang menyediakan ruang di server agar website dapat diakses melalui internet. Ini adalah konsep vital dalam Administrasi Sistem Jaringan.

Shared Hosting

Satu server dibagi-bagi untuk banyak pengguna. Biaya murah, cocok untuk website pemula atau kecil dengan traffic rendah.

VPS Hosting

Virtual Private Server memberikan sumber daya khusus dengan pembagian server secara virtual. Lebih fleksibel dan powerful.

Dedicated Server

Satu server fisik penuh hanya untuk satu pengguna. Performa tinggi, kontrol penuh, cocok untuk perusahaan besar.

Cloud Hosting

Menggunakan jaringan server yang saling terhubung. Sangat skalabel, dapat menyesuaikan beban lalu lintas secara dinamis.

Materi cPanel

Antarmuka cPanel
cPanel adalah panel kontrol hosting berbasis web yang memudahkan pengguna untuk mengelola website dan layanan hosting mereka melalui antarmuka grafis yang user-friendly.

Pengelolaan File

File Manager memungkinkan pengguna untuk mengunggah, mengedit, menghapus, dan mengelola file website secara langsung melalui browser.

Manajemen Database

cPanel menyediakan phpMyAdmin untuk mengelola database MySQL, termasuk membuat, mengedit, dan menghapus database serta tabel.

Manajemen Domain

Pengguna dapat menambahkan domain, subdomain, mengatur redirect, dan mengelola zona DNS melalui fitur Domain.

Manajemen Email

cPanel memungkinkan pembuatan akun email dengan domain sendiri, mengatur forwarder, autoresponder, dan mengakses webmail.

Keamanan

Fitur keamanan seperti SSL/TLS, IP Blocker, dan Hotlink Protection membantu melindungi website dari ancaman keamanan.

Instalasi Otomatis

Dengan Softaculous, pengguna dapat menginstal CMS seperti WordPress, Joomla, atau Drupal dengan satu klik.
